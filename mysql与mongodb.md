# 什么是 MongoDB
MongoDB是基于分布式存储的非关系型数据库，是一种文档型数据库，适合存储海量数据并提取高性能存取
三元素：1. 数据库、集合（表）、文档（行）

对于我这边来说，用的MongoDB比较多，一般都是采用nodejs的方式来连接数据库，
